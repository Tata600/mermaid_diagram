Mermaid Diagrams

Render beautiful diagrams in Knowledge articles

Features

  • Select "Mermaid" from code block language dropdown
  • Live diagram rendering
  • Light/Dark theme adaptive
  • Supports flowcharts, sequence, class, state, ER, Gantt, pie charts

Usage

  1. Insert a code block (/code)
  2. Select Mermaid from language dropdown
  3. Enter Mermaid syntax
  4. Diagram renders automatically

Example

flowchart TD
    A[Start] --> B{Decision}
    B -->|Yes| C[Done]
    B -->|No| A
